success criteria


new projects can be managed (crud)
video files can be added to a project. 
a timeline shows video files. 
the timeline can be seeked with a playhead and the video image is shown. 
the video can be played in realtime or 1x 2x 5x +- speeds. 

a video section can be created by: 
    pressing a keyboard shortcut (ctrl+m) will add a new playhead market at the current time of the playhead.
    a second playhead marker will create a new o_video_section. 
    a playhead marker can be deleted by (ctrl+d), the playhead marker that is closest to the current playhead will be deleted


each functionality has a name and a shortcut assigned, shortcuts can be changed in settings. 



a project can be exported to a video . this will be simply all o_video section clips concatenated.
